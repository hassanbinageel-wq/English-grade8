## 📊 تحليل الكتاب (نظرة المدرّس)

الكتاب يتدرّج وظيفياً (Functional syllabus) لا قواعدياً فقط، وكل وحدة تبني على ما قبلها:

1. **Unit 1 — Personal Details:** المضارع البسيط، like/don't like، have got، وصف البيت.
2. **Unit 2 — Getting Around & Shopping:** الماضي البسيط ونفيه (didn't)، would like، this/that/these/those، الاقتراحات (Let's / How about + ing).
3. **Unit 3 — Some / Any & Feelings:** some/any مع المعدود وغير المعدود، will لعرض المساعدة، صفات المشاعر، ضمائر الفاعل/المفعول.
4. **Unit 4 — Future, Directions & Past Continuous:** will / won't، إعطاء الاتجاهات، can/can't، الماضي المستمر (was/were + ing).
5. **Unit 5 — Camping, Comparing & Possessives:** أنشطة التخييم، How far، ضمائر الملكية (mine/yours…)، ترتيب التعليمات (first/next/then).
6. **Unit 6 — Animals & Superlatives:** وصف الحيوانات، أفعل التفضيل (the -est)، have to / has to.
7. **Unit 7 — Jobs & Family:** المهن، العلاقات العائلية، would like to be، How much / How many.

التطبيق صُمّم ليطابق هذا التدرّج: كل درس يجمع الشرح + المفردات + التطبيق + الاختبار في مكان واحد، ويُبرز نقاط الضعف ليعالجها الطالب أولاً بأول.

## 🛠️ التقنية

- واجهة ويب خفيفة (HTML/CSS/JS صرفة، بدون أي مكتبات خارجية → سريعة وتعمل أوفلاين).
- تُغلّف كتطبيق أندرويد عبر **Capacitor**.
- يُبنى ملف الـ APK **تلقائياً** عبر **GitHub Actions** (لا تحتاج Android Studio على جهازك).

```
www/                ← التطبيق (هو محتوى الـ APK)
 ├─ index.html
 ├─ css/style.css
 ├─ js/data.js      ← كامل محتوى الكتاب (الوحدات/الدروس/المفردات/الأسئلة)
 ├─ js/app.js       ← منطق التطبيق ومحرّك الألعاب
 ├─ manifest.json + sw.js   ← دعم العمل بدون إنترنت
 └─ assets/         ← الأيقونات
assets/             ← مصدر أيقونة وسبلاش الأندرويد (1024)
capacitor.config.json
package.json
.github/workflows/build-apk.yml   ← يبني الـ APK تلقائياً
```

## 🚀 الرفع على GitHub والحصول على الـ APK

### الطريقة الأسهل (موصى بها)

1. أنشئ مستودعاً جديداً على GitHub (مثلاً `misbah-english-grade8`).
2. من داخل مجلّد المشروع، نفّذ:

```bash
git init
git add .
git commit -m "مصباح الإنجليزية - الإصدار الأول"
git branch -M main
git remote add origin https://github.com/<اسم-حسابك>/misbah-english-grade8.git
git push -u origin main
```

3. بمجرّد الرفع، ينطلق **GitHub Actions** ويبني الـ APK تلقائياً (٥–١٠ دقائق).
4. اذهب إلى تبويب **Actions** لمتابعة البناء، ثم إلى تبويب **Releases**.
5. ستجد ملف **`Misbah-English-Grade8.apk`** جاهزاً للتنزيل.

> إن لم يظهر تبويب Releases: افتح **Actions → آخر تشغيل → Artifacts** ونزّل `Misbah-English-Grade8-APK`.

### تثبيت الـ APK على الهاتف
1. انقل الملف إلى هاتف أندرويد.
2. عند الفتح، فعّل **"السماح بالتثبيت من مصادر غير معروفة"**.
3. ثبّت التطبيق — يعمل كاملاً بدون إنترنت.

## 💻 البناء محلياً (اختياري)

يتطلّب: Node.js 18+، JDK 17، Android SDK.

```bash
npm install
npx cap add android
npx capacitor-assets generate --android   # توليد الأيقونات
npx cap sync android
cd android && ./gradlew assembleDebug
# الناتج: android/app/build/outputs/apk/debug/app-debug.apk
```

## 🌐 تجربته كموقع فوراً
